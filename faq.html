FAQ

🎯 基础概念类

Q1：为什么同一个期权昨天 0.16，今天就变成 0.14？

因为时间价值（Theta）每天在扣钱。
弱上涨不足以抵消时间衰减，所以 OTM 期权会自动变便宜。
便宜不是好事，是更接近归零。

Q2：今天买 0.14 比昨天买 0.16 更划算吗？

是的。
因为你花更少的钱承担同样的未来走势风险。
昨天买的人并没有任何优势，只是更早承担了时间损耗。

Q3：什么是远价外（OTM）？为什么便宜？

OTM = 当前价格离行权价很远。
便宜原因是市场认为它 很难涨到行权价。
价格低 ≠ 好机会，而是 低概率的反映。

Q4：什么是大阳线、大阴线？

  • 大阳线：强力上涨棒，代表买盘强势。

  • 大阴线：大幅下跌，代表卖压强势。
    强阳/强阴是你做 OTM 期权最重要的信号。

🎯 操作逻辑类

Q5：什么时候绝对不要买 OTM 期权？（禁区）

  • 横盘（Sideways）

  • 弱趋势 + 快到期

  • 重大事件前、IV 高（溢价昂贵）

横盘=最大亏钱地。

Q6：什么时候可以买？（弱趋势）

方向明确但不强时，
适合买 小额度练习单（$10–50），
目标收益 = 50%。

Q7：什么时候最值得买？（强趋势）

  • 强阳线、加速上涨

  • 强阴线、连续下杀

  • 放量

  • 下午趋势变强

  • Put/Call 倾向明显

这是 OTM 的 黄金时间，可以做大一点。

Q8：Call 和 Put 如何选择？

系统会根据 5 大数据评分：

  • Fear & Greed Index

  • Put/Call Ratio

  • VIX

  • 10Y Treasury Yield

  • SPY 当日走势强弱

得分 > 0 → Call（看涨）
得分 < 0 → Put（看跌）

Q10：到期日前涨到行权价就一定赚钱吗？

不一定，看你是否在上涨过程中卖出。

时间越接近到期 → 波动越大 → 风险越大。

Q11：如果不卖，到期日会怎样？

如果到期日仍然低于行权价（709）：

Call 期权价值 = 0（归零）

Q12：为什么 SPY 涨，你的期权反而跌？

三个可能：

  1. 涨幅不够（弱上涨）

  2. 时间价值今天扣得更多（Theta）

  3. 隐含波动率 IV 下跌（影响价格）

Q13：是否可以同时买 Call 和 Put？

叫“Straddle / Strangle”，非常高级，需要更高的判断力。
目前不建议新手用，因为：

  • 两边都亏的概率远大于你想象

  • 时间价值双倍扣

Q14：为什么散户亏钱？我为什么能比他们强？

散户失败的三点：

  1. 无系统，不看数据，凭感觉买

  2. 不懂 Theta（时间损耗）

  3. 盯盘情绪化，导致过度交易

你走的是“数据 + 系统化”模式 → 压倒大部分散户。
